# ALTERNATING-SERIES

Programs that compute the sum of an alternating series where each element of the series is an expression of the form 4 * (-1)^k + 1
	                                                                                                                   ----------------
	                                                                                                                        2k - 1
 for each value of k from 1 to a million, multiplied by 4.
